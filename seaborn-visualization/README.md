## Project Overview
This project analyzes monthly revenue trends across different customer segments (Premium, Standard, and Budget) for Wunsch Keeling's retail client. The visualization demonstrates seasonal patterns and growth trajectories throughout the year.

## Files Included
- `README.md` - This file with student information
- `chart.py` - Python script that generates the Seaborn lineplot
- `chart.png` - Generated visualization (512x512 pixels)

## Key Insights
The visualization reveals:
- Strong upward revenue trends across all customer segments
- Significant holiday season boost in November-December
- Premium segment shows highest growth and seasonal sensitivity
- Budget segment maintains steady growth with less volatility

## Technical Details
- **Library**: Seaborn (built on Matplotlib)
- **Chart Type**: Line plot with multiple segments
- **Styling**: Professional whitegrid style with custom color palette
- **Dimensions**: 512x512 pixels (8x8 inches at 64 DPI)

## How to Run
```bash
python chart.py
```

This will generate `chart.png` with the monthly revenue analysis visualization.
